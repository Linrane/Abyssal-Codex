## 游戏概述

深渊牌匣是一款终端Roguelike卡牌构筑游戏。玩家选择一位英雄，深入三层深渊，通过构筑牌组、收集遗物来击败最终Boss。

### 核心循环

```
选择英雄 → 地图探索 → 遭遇事件 → 战斗/商店/奇遇/休息 → Boss守卫 → 下一层 → 最终胜利
```

单局结构：3大层 × (4~6个小房间 + 1Boss房) ≈ 15~20场战斗 + 10+事件

死亡后获得「深渊记忆」点数，用于解锁新卡牌、新职业和永久强化。

### 游戏操作

| 按键 | 功能 |
|------|------|
| ↑↓←→ | 菜单导航 / 选牌 |
| Enter | 确认 |
| 1-9 | 快速选牌（战斗） |
| E | 结束回合（战斗） |
| D | 查看牌组 |
| Tab | 切换目标（战斗） |
| R | 重抽手牌（需蛇之蜕皮遗物） |
| L | 在主菜单切换中英文 |
| ESC | 返回上一步 / 退出 |

## 职业与英雄

### 深渊骑士（初始解锁）
- **HP**: 80 | **能量**: 3
- **核心机制**: 易伤、破甲
- **初始牌组**: 5打击，4防御，1重击
- **风格**: 利用易伤打出爆发伤害，适合新手

### 影织者（初始解锁）
- **HP**: 65 | **能量**: 3
- **核心机制**: 中毒、闪避
- **初始牌组**: 5打击，4防御，1毒刃
- **风格**: 叠毒消耗战，闪避规避伤害

### 符文贤者（解锁：累计击败10个Boss）
- **HP**: 60 | **能量**: 4
- **核心机制**: 充能、共鸣
- **风格**: 充能达到阈值触发连锁反应

### 血契者（解锁：单局使用20次能力牌）
- **HP**: 55 | **能量**: 3
- **核心机制**: 血怒
- **风格**: HP越低伤害越高，高风险高回报

### 流浪剑豪（隐藏职业）
- **HP**: 70 | **能量**: 3
- **核心机制**: 架势切换
- **风格**: 攻击/防御/疾风三种架势灵活切换

- **深渊审判** (3⚡): 全体25伤害+3层易伤
- **暗影之舞** (2⚡): 3层闪避，抽3，获得2能量
- **星核坍缩** (3⚡): 全体30伤害+冻结

## 深渊层介绍

### 第一层：崩塌圣堂
古老神殿在月光的照耀下沉睡，断裂的柱廊间游荡着被虚空侵蚀的圣骑士。

**地形效果**: 每场战斗开始时双方获得1层易伤。

### 第二层：幽暗密林
扭曲的森林深处，沼泽散发着令人窒息的有毒雾气。暗影中潜伏着不知名的猎手。

**地形效果**: 第3回合起，全场获得1层中毒。

### 第三层：星核裂隙
虚空浮岛漂浮在星辰之海中。时空的法则在这里被扭曲，能量变得不稳定。

**地形效果**: 能量回复+1，但卡牌费用随机波动±1。

## 事件与奇遇

- **回响之泉**: 选择移除一张牌 OR 获得一张稀有牌+一张诅咒
- **封印之箱**: 支付50金币获得遗物 OR 40%概率获得遗物（60%受伤）
- **共鸣之泉**: 回复25%生命 OR 升级一张随机卡牌
- **碎镜之间**: 移除所有诅咒牌 OR 随机转化2张卡牌

## 攻略与技巧

### 新手建议
1. **优先走精英路径** — 精英掉落更丰厚的奖励
2. **保持牌组精简** — 移除基础打击/防御牌，保持牌组20-25张
3. **善用休息点** — 升级关键牌比回血更优先
4. **关注敌方意图** — 预判伤害，合理安排格挡

### 职业攻略

#### 深渊骑士
- 优先获取高伤害攻击牌和易伤来源
- 组合「撕裂」+「重击」可打出爆发伤害
- 「铁壁」提供荆棘反击

#### 影织者
- 堆叠中毒是主要输出手段
- 「暗影步」提供闪避和抽牌
- 配合「毒液瓶」遗物可稳定叠毒

### 遗物组合
- **血之护符 + 铁鳞甲**: 每战开局优势
- **毒液瓶 + 影织者**: 每回合自动叠毒
- **远古典籍 + 任何职业**: 多抽牌=多选择
- **幸运金币 + 商店路线**: 经济优势
